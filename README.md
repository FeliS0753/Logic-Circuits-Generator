## 🛠 環境構築手順
ローカル環境で動作させる場合は、以下の手順に従ってください。

```bash
# リポジトリをクローン
git clone https://github.com

# ディレクトリに移動
cd repository

# パッケージのインストール
npm install

# 環境変数の設定
cp .env.example .env
# ※ .envの中身を適宜修正してください

# 起動
npm run dev
```
